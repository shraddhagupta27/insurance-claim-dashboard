# Insurance & Claims Analysis Dashboard (Power BI)

### Project Overview
An interactive Power BI dashboard analyzing insurance policy and claim data to help identify risk patterns, customer segments, and financial performance drivers.

### Objectives
- Centralize policyholder and claims data for visibility across departments  
- Track KPIs like Total Policies, Total Claim Amount, Claim Frequency, and Average Claim Value
- Enable segmentation by car use, education level, marital status, region, and age group
- Support risk assessment and policy optimization through interactive analysis  

### Tools & Techniques
- Power BI:Dashboard design, DAX measures, data modeling   

### Skills Demonstrated
Power BI • DAX • KPI Design • Data Modeling • Financial Analytics • Risk Analysis • Data Storytelling
